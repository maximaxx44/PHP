Лабораторная работа №3. Управляющие конструкции

Цель работы

Освоить использование условных конструкций и циклов в PHP.

Ход выполнения работы

Сначала я запустил XAMPP Control Panel и включил модуль Apache, чтобы локальный сервер начал работать.

Далее я вставил в файл index.php программный код лабораторной работы на PHP.

Основная задача программы - сформировать расписание работы сотрудников в зависимости от текущего дня недели, а также продемонстрировать работу различных видов циклов в PHP.

В программе определяется текущий день недели с помощью функции date():

$day = (int)date('N');

Функция date() возвращает текущую дату или время в заданном формате. Формат 'N' используется для получения номера дня недели. Значение возвращается в диапазоне от 1 до 7, где:

1 - понедельник

2 - вторник

3 - среда

4 - четверг

5 - пятница

6 - суббота

7 - воскресенье

Оператор (int) выполняет приведение типа, преобразуя результат функции в целое число. Полученное значение сохраняется в переменной $day.

После этого в программе задаются значения графика работы сотрудников по умолчанию:

$johnSchedule = "Нерабочий день";
$janeSchedule = "Нерабочий день";

Здесь создаются переменные $johnSchedule и $janeSchedule, в которых будет храниться график работы сотрудников. Изначально им присваивается значение «Нерабочий день», которое будет изменено в случае, если текущий день является рабочим.

Далее используется условная конструкция if, которая определяет рабочие дни сотрудника John Styles:

if ($day == 1 || $day == 3 || $day == 5) {
    $johnSchedule = "8:00-12:00";
}

Условие проверяет, равен ли текущий день недели значениям 1, 3 или 5, что соответствует понедельнику, среде и пятнице. Оператор == используется для сравнения значений, а оператор || обозначает логическое ИЛИ. Если условие выполняется, переменной $johnSchedule присваивается значение «8:00–12:00», обозначающее рабочие часы сотрудника.

Аналогичным образом определяется график работы сотрудника Jane Doe:

if ($day == 2 || $day == 4 || $day == 6) {
    $janeSchedule = "12:00-16:00";
}

Данная конструкция проверяет, является ли текущий день вторником, четвергом или субботой. Если условие истинно, переменной $janeSchedule присваивается значение «12:00–16:00», которое соответствует рабочему времени сотрудника.

Для удобства отображения названия текущего дня недели в программе создаётся массив:

$daysRu = [
    1 => "Понедельник",
    2 => "Вторник",
    3 => "Среда",
    4 => "Четверг",
    5 => "Пятница",
    6 => "Суббота",
    7 => "Воскресенье"
];

Массив $daysRu хранит соответствие между номером дня недели и его названием на русском языке. Каждый элемент массива состоит из ключа (номер дня недели) и значения (название дня).

Далее определяется название текущего дня:

$dayName = $daysRu[$day] ?? "Неизвестно";

В данном выражении происходит обращение к элементу массива $daysRu по индексу $day. Оператор ?? называется оператором объединения с null и используется для проверки существования значения. Если элемент массива существует, используется его значение, иначе выводится строка «Неизвестно».

Во второй части программы реализована демонстрация работы циклов. Сначала создаётся функция:

function run_for(): array {

Функция run_for() предназначена для выполнения цикла for. Запись : array означает, что функция возвращает массив.

Внутри функции задаются начальные значения переменных:

$a = 0;
$b = 0;

Переменные $a и $b используются для демонстрации изменения значений в цикле. Изначально им присваивается значение 0.

Далее создаётся массив:

$log = [];

Этот массив предназначен для хранения промежуточных результатов выполнения цикла.

После этого запускается цикл for:

for ($i = 0; $i <= 5; $i++) {

Цикл for состоит из трёх частей:

Инициализация переменной счётчика $i = 0;

Условие выполнения цикла $i <= 5;

Изменение значения счётчика $i++.

Цикл выполняется до тех пор, пока значение $i меньше или равно 5.

Внутри цикла выполняются операции:

$a += 10;
$b += 5;

Оператор += является сокращённой формой записи сложения.
Например, выражение $a += 10 эквивалентно записи $a = $a + 10.
Таким образом, при каждой итерации значение $a увеличивается на 10, а значение $b увеличивается на 5.

Затем формируется строка с промежуточными результатами:

$log[] = "Step $i: a = $a, b = $b";

Оператор $log[] добавляет новый элемент в конец массива $log. В массив записывается строка, содержащая номер шага и текущие значения переменных.

После завершения цикла формируется итоговое сообщение:

$end = "End of the loop: a = $a, b = $b";

Данная строка содержит окончательные значения переменных $a и $b.

В завершение функция возвращает результат:

return [$log, $end];

Функция возвращает массив, содержащий список промежуточных значений и итоговый результат выполнения цикла.
Контрольные вопросы:

1. В чем разница между циклами for, while и do-while? В каких случаях лучше использовать каждый из них?

Цикл for используется, когда заранее известно количество повторений, так как в нём удобно задаются начальное значение счётчика, условие и шаг изменения. Циклы while и do-while применяются, когда количество повторений заранее неизвестно, при этом while проверяет условие перед выполнением, а do-while — после выполнения тела цикла.

2. Как работает тернарный оператор ?: в PHP?

Тернарный оператор — это сокращённая форма условного оператора if...else, позволяющая записать проверку условия в одну строку. Если условие истинно, возвращается первое значение, если ложно — второе.

3. Что произойдёт, если в do-while поставить условие, которое изначально ложно?
   
В цикле do-while тело цикла выполняется как минимум один раз, потому что проверка условия происходит после выполнения команд внутри цикла. Поэтому даже если условие изначально ложно, код внутри цикла выполнится один раз, после чего цикл завершится.
